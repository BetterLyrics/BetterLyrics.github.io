title: 应用外观和行为

### 应用外观
此部分包含应用程序的显示语言设置。

* **路径**：`设置` -> `应用外观和行为` -> `应用外观` -> `语言`
    * **说明**：更改应用程序的显示语言。更改后需要点击下方的 **重启应用以应用更改** 按钮，或者手动重启应用生效。
    * **附加功能**：
        * **重启应用以应用更改**：立即重启应用以应用新的语言设置。
        * **帮助我们翻译此应用程序**：跳转到 Crowdin 翻译平台，参与应用的本地化工作。

### 启动
此部分控制应用程序随系统启动的行为以及启动时的初始状态。

1. **路径**：`设置` -> `应用外观和行为` -> `启动` -> `自动启动`
  * **说明**：开启后，BetterLyrics 将随 Windows 系统启动而自动运行。

2. **路径**：`设置` -> `应用外观和行为` -> `启动` -> `启动时打开音乐库窗口`
    * **说明**：开启后，每次启动程序时会自动弹出“音乐库”界面。

3. **路径**：`设置` -> `应用外观和行为` -> `启动` -> `启动时自动打开默认歌词窗口`
    * **说明**：开启后，程序启动时会自动加载并显示歌词悬浮窗（Desktop/Docked 等模式）。

### 音乐库窗口
此部分配置与“音乐库”窗口交互时的自动化行为。

* **路径**：`设置` -> `应用外观和行为` -> `音乐库` -> `打开音乐库窗口时自动继续播放`
    * **说明**：当您打开音乐库界面时，如果当前处于暂停状态，会自动开始播放音乐。

* **路径**：`设置` -> `应用外观和行为` -> `音乐库` -> `关闭音乐库窗口时停止播放`
    * **说明**：当您关闭音乐库界面（点击 X）时，会自动停止当前的音乐播放。

* **路径**：`设置` -> `应用外观和行为` -> `音乐库` -> `关闭音乐库窗口时退出程序`
    * **说明**：当您关闭音乐库界面时，不仅仅是关闭窗口，而是彻底退出 BetterLyrics 应用程序。

### 歌词窗口
此部分配置关闭歌词悬浮窗时的行为。

* **路径**：`设置` -> `应用外观和行为` -> `歌词窗口` -> `关闭歌词窗口时退出程序`
    * **说明**：当您手动关闭歌词悬浮窗时，彻底退出 BetterLyrics 应用程序。

### 快捷键
此部分允许用户自定义全局快捷键以控制程序行为。

* **路径**：`设置` -> `应用外观和行为` -> `快捷键` -> `歌词窗口显示和隐藏快捷键`
    * **说明**：设置一个组合键，用于快速隐藏或呼出歌词悬浮窗。

* **路径**：`设置` -> `应用外观和行为` -> `快捷键` -> `歌词窗口状态切换快捷键`
    * **说明**：设置一个组合键，用于在不同的歌词窗口模式（如桌面模式、停靠模式等）之间进行切换。

* **路径**：`设置` -> `应用外观和行为` -> `快捷键` -> `播放与暂停快捷键`
    * **说明**：设置控制媒体播放/暂停的全局快捷键。

* **路径**：`设置` -> `应用外观和行为` -> `快捷键` -> `下一曲目快捷键`
    * **说明**：设置切歌到下一首的全局快捷键。

* **路径**：`设置` -> `应用外观和行为` -> `快捷键` -> `上一曲目快捷键`
    * **说明**：设置切歌到上一首的全局快捷键。
