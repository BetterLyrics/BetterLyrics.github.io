## 从侧载包安装

:::caution[提示]
操作前请保存当前正在进行的工作
:::

### 方式 1：通过 PowerShell 脚本安装

<Steps>

1. 解压缩下载的 `.zip` 文件，右键单击 `install.ps1`，选择 `使用 PowerShell 运行`。

2. 按 `回车` 键继续，并同意弹出窗口中的内容。

3. 输入 `Y` 安装证书。

	:::caution[提示]
	等待程序安装完成，该步骤可能耗费较长时间，且会出现其他程序自行退出或重启的情况，请在此操作前保存当前的工作。
	:::

	现在一切就绪。

</Steps>

_注：如果在已安装旧版本的情况下安装失败，请尝试卸载旧版本并重新安装。_

### 方式 2：半自动安装

#### 手动安装证书

双击打开 `.cer` 证书文件，跟随以下操作：

<Steps>

1. `安装证书(I)...`
2. `本地计算机(L)`
3. `下一步(N)`
4. `将所有的证书都放入下列存储(P)`
5. `浏览(R)...`
6. `受信任的根证书颁发机构`
7. `确定`
8. `下一步(N)`
9. `完成(F)`

</Steps>

#### 安装程序包

双击打开 `.msixbundle` 安装包，单击	`安装` 即可。
