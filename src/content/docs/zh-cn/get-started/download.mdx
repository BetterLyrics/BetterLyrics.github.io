## Microsoft Store

BetterLyrics 已经上架 Microsoft Store，您可以直接在 Microsoft Store 中搜索 "BetterLyrics" 来下载和安装最新版本的 BetterLyrics。

[点击这里访问 Microsoft Store 上的 BetterLyrics 页面](https://apps.microsoft.com/detail/9p1wcd1p597r)

如果您更喜欢从 GitHub Releases 页面下载，您也可以访问以下链接：

如果您是开发者或者想要从源代码构建 BetterLyrics，您可以克隆这个仓库并使用 Visual Studio 来构建项目。

```bash
git clone https://github.com/jayfunc/BetterLyrics.git
```
