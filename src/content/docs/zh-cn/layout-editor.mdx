欢迎使用 BetterLyrics 自定义布局编辑器！通过这个强大的可视化工具，你可以自由设计歌词窗口的排版。无论是调整网格、拖拽组件，还是精细化设置属性，都可以通过直观的交互轻松完成。

## 画布与网格管理

布局的基础是 **网格系统**。你可以将整个歌词窗口想象成一个可以自由划分行和列的表格。

### 全局设置
在左侧的 “画布设置” 区域，你可以控制整个网格的基础形态：
* **添加/移除行列**：使用顶部的按钮可以快速在网格末尾追加或移除一整行/列。
* **行列间距**：调整全局的行距和列距，让组件之间保持呼吸感。
* **画布内边距**：分别设置画布的上、下、左、右内边距，控制内容与窗口边缘的距离。

### 快捷网格操作（上下文菜单）
在中间画布区域的 **行标题（左侧数字）** 或 **列标题（顶部数字）** 上点击或右键，可以唤出高级菜单：
* **插入**：在当前行/列的前面或后面插入新的行/列。
* **删除**：直接删除选中的行/列。
* **切换尺寸模式**：点击星号图标，可以在 “自适应大小” 和 “比例大小” 之间切换。

## 组件的添加与图层管理

界面左侧包含了你的 “组件工具箱” 和 “图层列表”。

* **待添加组件**：这里列出了所有可用的界面元素。只需 **按住组件并拖拽** 到中间的画布网格中，即可将其放置在对应位置。
* **已放置组件**：这里以列表形式展示了画布上现有的所有组件。点击列表中的项，可以快速在画布中选中它，这在组件相互重叠或被遮挡时非常有用。

## 组件属性与调整

当你在画布中点击选中某个组件后，它会被高亮显示（带有蓝色边框和调整手柄），右侧的 “属性面板” 将随之激活。

### 调整位置与大小
* **拖拽移动**：直接按住选中的组件并拖动，即可将其移动到新的网格单元格中。
* **手柄拉伸**：拖动组件右侧、底部或右下角的调整手柄，可以快速改变它占据的行数和列数。
* **精确数值**：你也可以在右侧属性面板中，手动输入 `起始行`、`起始列`、`跨行数` 和 `跨列数`。

### 对齐与边距
* **外边距**：在属性面板中设置组件的上、下、左、右外边距，用于微调它在网格单元格内的精确位置。
* **对齐方式**：支持水平（左、中、右、拉伸）和垂直（顶、中、底、拉伸）对齐方式，决定组件如何在单元格内停靠。

## 画布视图控制

为了方便编辑复杂的布局，顶部工具栏提供了一系列视图控制功能：

* **抓手工具**：点击手掌图标激活。激活后，拖动画布即可自由平移视角。
* **放大 / 缩小**：使用放大镜图标调整画布比例，也可以点击“重置缩放”让画布自动适应当前可用空间。

## 键盘快捷键速查

熟练使用快捷键可以大幅提升你的布局搭建效率。按下 `F1` 或 `Ctrl + /` 也可以在编辑器内随时唤出此帮助菜单。

### 视图导航
| 操作 | 快捷键 |
| :--- | :--- |
| **临时激活抓手工具** | 长按 `空格键` |
| **缩放画布** | `Ctrl` + 鼠标滚轮 |
| **水平滚动** | `Shift` + 鼠标滚轮 |

### 组件操作
| 操作 | 快捷键 |
| :--- | :--- |
| **移动选中组件** | 方向键 或 `W` `A` `S` `D` |
| **改变组件跨度（大小）** | `Shift` + 方向键 或 `W` `A` `S` `D` |
| **删除选中组件** | `Delete` 或 `Backspace` |
| **取消选中** | `Esc` |

---
*提示：所有的修改都会实时在画布中预览，你可以随时拖拽、调整并观察最终效果。*
