## Path: Settings -> App Appearance & Behaviors

### App Appearance
This section contains the display language settings for the application.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `App Appearance` -> `Language`
    * **Description**: Change the application's display language. After changing, you need to click the **Restart app to apply changes** button below, or manually restart the app for the changes to take effect.
    * **Additional Features**:
        * **Restart app to apply changes**: Immediately restarts the app to apply the new language settings.
        * **Help translate this app**: Redirects to the Crowdin translation platform to participate in the localization of the app.

### Startup
This section controls how the application behaves when the system starts and its initial state upon launch.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Startup` -> `Auto Start`
    * **Description**: When enabled, BetterLyrics will automatically run when Windows starts.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Startup` -> `Open Music Library window on startup`
    * **Description**: When enabled, the "Music Library" interface will automatically pop up every time the program starts.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Startup` -> `Automatically open default lyrics window on startup`
    * **Description**: When enabled, the program will automatically load and display the lyrics floating window (Desktop/Docked modes, etc.) upon launch.

### Music Library Window
This section configures automation behaviors when interacting with the "Music Library" window.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Music Library` -> `Automatically resume playback when opening Music Library window`
    * **Description**: When you open the Music Library interface, if playback is currently paused, it will automatically resume playing music.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Music Library` -> `Stop playback when closing Music Library window`
    * **Description**: When you close the Music Library interface (clicking X), it will automatically stop the current music playback.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Music Library` -> `Exit application when closing Music Library window`
    * **Description**: When you close the Music Library interface, it will not just close the window but completely exit the BetterLyrics application.

### Lyrics Window
This section configures the behavior when closing the lyrics floating window.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Lyrics Window` -> `Exit application when closing Lyrics Window`
    * **Description**: When you manually close the lyrics floating window, it completely exits the BetterLyrics application.

### Shortcuts
This section allows users to customize global shortcuts to control program behavior.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Shortcuts` -> `Show/Hide Lyrics Window shortcut`
    * **Description**: Set a key combination to quickly hide or summon the lyrics floating window.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Shortcuts` -> `Toggle Lyrics Window state shortcut`
    * **Description**: Set a key combination to switch between different lyrics window modes (such as Desktop mode, Docked mode, etc.).

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Shortcuts` -> `Play/Pause shortcut`
    * **Description**: Set a global shortcut to control media playback/pause.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Shortcuts` -> `Next Track shortcut`
    * **Description**: Set a global shortcut to skip to the next song.

* **Path**: `Settings` -> `App Appearance & Behaviors` -> `Shortcuts` -> `Previous Track shortcut`
    * **Description**: Set a global shortcut to go back to the previous song.
